# Cache
In order to download, compile and run the LRU_cache:
```
git clone https://github.com/aywert/Cache
cd Cache
make Release
make run_lru
```
For the further use just run:

```
make run_lru
```
For running tests of LRU cache:
```
make Debug
make test_lru
```

In order to run the OPT_cache:
```
make run_opt
```
For running tests of LRU cache:
```
make Debug
make test_opt
```
